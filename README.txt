== Description ==
Tookapic Photo Feed Plugin is a simple widget for Tookapic users who wants share their photos on own WordPress blog or website.

== Frequently Asked Questions ==

= Do I need account on Tookapic? =
Yes. You need to have an account on [Tookapic](https://tookapic.com)

= Installation Instructions =
1. Download the plugin .zip file.
2. In the WordPress admin go to Plugins > Add New or click the “Add New” button on the main plugins screen.
3. On the following screen, click the “Upload Plugin” button.
4. Browse your computer to where you downloaded the plugin .zip file, select it and click the “Install Now” button.
5. After the plugin has successfully installed, click “Activate Plugin” and enjoy!

= After Installation =
After installation, you will have new Widget in widget menu named "Tookapic". Just add a widget to your sidebar and fill the inputs. Required is only your username. Save the widget and go to your website. If everything will be fine, you will be satisfied plugin user with your current photos from Tookapic at your website.
